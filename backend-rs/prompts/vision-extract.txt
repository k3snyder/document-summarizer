1. **EXACT VALUES ONLY**
   ✅ "Q1: $42,500, Q2: $51,200"
   ❌ "Sales showing growth trend"

2. **PRESERVE ORIGINAL TEXT**
   ✅ Copy labels, titles verbatim
   ❌ Paraphrase or interpret

3. **NEVER HALLUCINATE**
   ✅ "[UNCLEAR: ~20-25%]" or omit
   ❌ Inventing plausible numbers

4. **COUNT AFTER LISTING**
   ✅ "Nodes: [list all]. Total: 4"
   ❌ "The four components are..."

5. **STRUCTURE OVER PROSE**
   ✅ Node-edge tables for diagrams
   ❌ Paragraph descriptions of connections

6. **REPORT, DON'T INTERPRET**
   ✅ "Red line shows 15% increase"
   ❌ "Company is performing well"

7. **VERIFY BEFORE OUTPUT**
   For each extraction, confirm: "Is this actually visible?" → YES required

8. **CAPTURE ALL TEXT IN DIAGRAMS** (commonly missed!)
   ✅ Extract text boxes: "Stateless operations. No customer data persisted."
   ✅ Extract legends: "🔒 Connections secured by HTTPS"
   ✅ Extract step descriptions: "1. User initiates request from application"
   ✅ Extract notes/disclaimers within diagram boundaries
   ❌ Only capturing node labels while ignoring callout boxes and annotations

9. **OCR SCANNED TEXT PAGES**
   ✅ For scanned book/report pages, transcribe the visible prose text even when there are no charts or diagrams
   ✅ Preserve paragraph order and visible headings
   ❌ Returning "No significant visual elements detected" when the page contains readable text
</critical_rules>
